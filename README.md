# GeoMoment Diary

GeoMoment Diary adalah aplikasi Android yang berfungsi sebagai buku harian digital. Pengguna dapat menyimpan momen penting dalam bentuk teks, foto, serta lokasi geografis tempat momen tersebut terjadi.

## Fitur Sesuai Spesifikasi UAS

- ✅ CRUD (Create, Read, Update, Delete) data catatan
- ✅ Penyimpanan data menggunakan Room Database
- ✅ Mengambil foto langsung menggunakan kamera
- ✅ Menampilkan preview foto pada form input
- ✅ Menyimpan path foto ke database
- ✅ Mendeteksi koordinat GPS secara otomatis
- ✅ Menyimpan koordinat Latitude dan Longitude

## Teknologi

- Kotlin
- Jetpack Compose
- Room Database
- Google Maps SDK
- Fused Location Provider

 ## Struktur Fitur

- Home
- Tambah Catatan
- Edit Catatan
- Detail Catatan
- Daftar Catatan
- Peta Momen
